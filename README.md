## 🚀 阶段一：Geek 的出厂设置 - `GitHub` 入门与 Markdown 学习

> 💡 **学习目标**：打开 GitHub 大门，学会用 Markdown 写优雅文档。

- [ ] 📝 了解 GitHub 是什么，注册账号并完善个人资料  
- [ ] 🐙 创建第一个仓库（Repository），初始化 README.md  
- [ ] ✨ 学习 Markdown 基础语法：标题、列表、链接、图片、代码块  
- [ ] 🎀 用 Markdown 写一份自我介绍（带表情包和可爱元素）  
- [ ] 🔖 掌握 Issue / Pull Request 的基本概念（看文档+实操）  


---

## ⚙️ 阶段二：Geek 的出厂设置 - `GitHub` 进阶与 **Git 学习**

> 💡 **学习目标**：用 Git 版本控制神器，玩转 GitHub 协作流程。

- [ ] 🧩 本地安装 Git，配置用户名和邮箱  
- [ ] 🌿 学习基本 Git 命令：`clone`, `add`, `commit`, `push`, `pull`  
- [ ] 🔀 理解分支（branch）和合并（merge），模拟团队合作  
- [ ] 🧸 在 GitHub 上 Fork 一个开源项目，并提交你的第一个 Pull Request  
- [ ] 📌 学会使用 `.gitignore` 忽略不需要的文件  

**🐣 小彩蛋**：用 `git log --oneline --graph --all` 画一棵可爱的提交树 🌲

---

## 🎨 阶段三：Web 相关领域的基础 - HTML / CSS / JavaScript

> 💡 **学习目标**：搭建网页三件套，让你的创意在浏览器中绽放。

- [ ] 🏗️ HTML 结构：语义化标签、表单、多媒体嵌入  
- [ ] 🎨 CSS 魔法：选择器、盒模型、Flexbox / Grid 布局  
- [ ] 🧸 给页面加一点“可爱风”：圆角、阴影、渐变色、emoji 字体  
- [ ] ⚡ JavaScript 入门：变量、函数、DOM 操作、事件监听  
- [ ] 🖼️ 做一个互动小作品（比如：点击按钮变出小星星✨）  

**🐣 小彩蛋**：用 CSS 画一只懒洋洋的小猫或小熊猫 🐼

---

## 🌈 阶段四：承上启下 & 大展身手

> 💡 **学习目标**：整合前三阶段技能，完成一个完整的个人项目，开启开源之路。

- [ ] 🧠 头脑风暴一个有趣的项目点子（个人博客 / 小工具 / 宠物相册）  
- [ ] 🛠️ 使用 Git + GitHub 管理项目代码，写好 README 和文档  
- [ ] 🌐 用 HTML/CSS/JS 实现原型，并部署到 GitHub Pages 上  
- [ ] 🤝 分享你的作品到社区（比如 V2EX、掘金、朋友圈），收集反馈  
- [ ] 🎉 回顾学习过程，写下你的“Geek 养成记”总结（Markdown 格式）  

**🐣 小彩蛋**：给你的项目起一个可爱的名字，比如“小饼干笔记簿”🍪
